# Загрузчик медиа

Программа для скачивания видео и аудио с YouTube, Rutube, VK, Twitch и др.

---

## Возможности

- 🎬 Скачивание видео в Full HD (1080p)
- 🎵 Конвертация в MP3 (192 kbps)
- 🌐 Поддержка YouTube, Rutube, VK, Twitch и др.
- 🖥️ Графический интерфейс
- 🇷🇺 Обход блокировок YouTube в России (автоповторы, прогрев соединения)

---

## Установка

### Windows

1. Скачай и установи Python 3.8+
   - ✅ Обязательно отметь галочку **«Add Python to PATH»** при установке

2. Запусти `Установщик_зависимостей.exe` (или `python depinstalator.py`)

   Установщик проверит и установит:
   - `yt-dlp` (библиотека загрузки)
   - `ffmpeg` (для Full HD видео и MP3)
   - Добавит `ffmpeg` в `PATH`

3. Запусти `Загрузчик_медиа.exe`

### macOS

1. Скачай и распакуй архив с программой

2. Запусти установщик зависимостей:

  python depinstalator.py

  Установщик проверит и установит:
  - Homebrew (если нет)
  - Python 3.12 (если системный устарел)
  - yt-dlp (библиотека загрузки)
  - ffmpeg (для Full HD видео и MP3)
3. Запусти `MediaDownloader.dmg`
